About
this project is a cybersecurity mini-project focused on performing a complete security audit of a test network. It combines custom Python scripts and industry-standard tools to identify vulnerabilities, analyze system security, and simulate real-world attacks such as scanning, crawling, and password testing.

# SecureScope - Security Audit Toolkit

## 📌 Project Overview

SecureScope is a modular cybersecurity toolkit developed in Python for performing security auditing and reconnaissance tasks. The project combines custom scripts and industry-standard tools to demonstrate real-world cybersecurity concepts in a controlled environment.

---
